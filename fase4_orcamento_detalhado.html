Observações sobre Hardware:

  • Os preços das câmeras variam muito com a resolução, FPS e fabricante. Os valores apresentados são estimativas para modelos que atendem aos requisitos mínimos.
  • O Jetson Xavier NX é o kit de desenvolvimento. Para uma solução de produção, o módulo embarcado e um gabinete industrial seriam necessários, aumentando o custo.
  • O módulo I/O digital industrial é uma alternativa mais robusta ao Raspberry Pi Pico para acionamento de alertas, mas com custo mais elevado.

Observações sobre Custos Azure:

  • A maior parte da inferência será feita no dispositivo Edge, o que reduzirá significativamente os custos de previsão do Custom Vision na nuvem.
  • Os custos de treinamento do Custom Vision são pontuais (durante o desenvolvimento e retreinamentos periódicos).
  • É altamente recomendável usar a calculadora de preços do Azure para uma estimativa mais precisa com base no uso esperado.

Análise de Custos:

  • O custo inicial de hardware é um investimento significativo, mas é um custo único.
