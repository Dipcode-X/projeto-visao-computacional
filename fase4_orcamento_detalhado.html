Fase 4: Orçamento Detalhado e Análise de Custos

Esta seção apresenta uma estimativa de custos para a implementação do sistema de visão computacional, baseada na pesquisa de hardware e nos serviços Azure. Os preços são aproximados e podem variar significativamente dependendo do fornecedor, da região, das especificações exatas do modelo e de negociações.

Observações sobre Hardware:

  • Os preços das câmeras variam muito com a resolução, FPS e fabricante. Os valores apresentados são estimativas para modelos que atendem aos requisitos mínimos.
  • O Jetson Xavier NX é o kit de desenvolvimento. Para uma solução de produção, o módulo embarcado e um gabinete industrial seriam necessários, aumentando o custo.
  • O módulo I/O digital industrial é uma alternativa mais robusta ao Raspberry Pi Pico para acionamento de alertas, mas com custo mais elevado.

2. Serviços Azure (Estimativa Mensal)

Os custos dos serviços Azure são baseados no consumo e podem variar significativamente. As estimativas abaixo são para um cenário de uso contínuo, considerando a inspeção de 12-20 caixas/minuto.

Serviço Azure Descrição Custo Mensal Estimado (USD)
Azure Custom Vision Treinamento e armazenamento de imagens. O custo varia com o número de imagens, tags e horas de treinamento. $50 - $300
Previsões (inferência) na nuvem (se usado como fallback ou para validação) $10 - $100
Azure IoT Hub Conectividade e mensagens entre o Edge e a nuvem. O custo depende do tier (Standard/Basic) e do número de mensagens. $25 - $150
Azure Functions Para processar mensagens do IoT Hub e enviar notificações. O custo é baseado no consumo (execuções e memória). $5 - $50
Azure Cosmos DB / SQL DB Armazenamento de logs e dados de não conformidade. O custo varia com o throughput e armazenamento. $30 - $200
Total Mensal Estimado (Azure) $125 - $820

Observações sobre Custos Azure:

  • A maior parte da inferência será feita no dispositivo Edge, o que reduzirá significativamente os custos de previsão do Custom Vision na nuvem.
  • Os custos de treinamento do Custom Vision são pontuais (durante o desenvolvimento e retreinamentos periódicos).
  • É altamente recomendável usar a calculadora de preços do Azure para uma estimativa mais precisa com base no uso esperado.
